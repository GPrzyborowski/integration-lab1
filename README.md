# SHA-256 Generator - integracja systemów informatycznych lab 1
**Celem laboratorium jest zapoznanie się z podstawami systemu kontroli wersji git oraz konfiguracja środowiska projektowego - w tym przypadku dla technologii Node.js. Wersja dla zadania nr 5**

## Wymagania
* zainstalowany system kontroli wersji git
* zainstalowany node oraz npm
* skonfigurowane konto Github
* wygenerowany klucz SSH do Github
* utworzone repozytorium projektu

Przed uruchomieniem projektu należu zainstalować wymagane zależności poleceniem ```npm i```

Aplikację można uruchomić poleceniem ```npm start```

[Oficjalna dokumentacja Node.js](https://nodejs.org/docs/latest/api/)

| Narzędzie | Komenda | Opis |
| :--- | :--- | :--- |
| **Git** | `git config --global user.name "imie"` | Konfiguracja tożsamości |
| **Node.js** | `node --version` | Sprawdzenie instalacji Node.js |
| **npm** | `npm init -y` | Inicjalizacja projektu |
| **npm** | `npm install express` | Instalacja frameworka |
| **Node.js** | `node index.js` | Uruchomienie aplikacji |
